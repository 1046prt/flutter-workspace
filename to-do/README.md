# To-Do App

A Flutter-based to-do list application.

## How to Run

**⚠️ IMPORTANT:** This is a Flutter app, not a plain Dart program.

### Run the app:

```bash
flutter run
```

### Or use VS Code:

- Press `F5` or click "Run and Debug"
- Select "Flutter: Run on Chrome" or "Flutter: Run on Device"

### ❌ DO NOT use:

```bash
dart lib/main.dart  # This will fail!
```

## Features

- Add new tasks
- Mark tasks as complete
- Delete tasks
- Clean and modern UI
