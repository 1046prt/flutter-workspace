# Calculator App

A Flutter-based calculator application.

## How to Run

**⚠️ IMPORTANT:** This is a Flutter app, not a plain Dart program.

### Run the app:

```bash
flutter run
```

### Or use VS Code:

- Press `F5` or click "Run and Debug"
- Select "Flutter: Run on Chrome" or "Flutter: Run on Device"

### ❌ DO NOT use:

```bash
dart lib/main.dart  # This will fail!
```

## Features

- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Clean and intuitive UI
- Responsive design
